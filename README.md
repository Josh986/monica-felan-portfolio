# Monica Felan — Pilates Instructor Website

A clean, elegant, and fully responsive static website for Monica Felan, certified Pilates instructor.

## Pages & Sections
- **Hero** — Full-screen introduction with CTA buttons
- **About** — Bio, certifications, and credentials
- **Services** — Private sessions, small group classes, online coaching, workshops
- **Philosophy** — Four core pillars of Monica's teaching method
- **Testimonials** — Client reviews
- **Contact** — Inquiry form + contact details

## Tech Stack
- Pure HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts: Cormorant Garamond + Montserrat
- Fully responsive (mobile-first)
- Deployed via GitHub Pages

## Deployment
This site is deployed via **GitHub Pages** from the `main` branch.
Live URL: `https://josh986.github.io/monica-felan-portfolio/`
